# MTA SAP Enterprise Messaging UI5 Websockets
A Multi Target Application containing a Node.js application connected to SAP Enterprise Messaging, serving as a Websocket for a Real-time UI5 Application. Logging all errors from the queue in real-time in the UI5 App.

It also contains some code examples of the very well explained SAP HANA Academy - SAP Cloud Platform Enterprise Messaging - Node.js consumer tutorial.

Their repository can be found here:
https://github.com/saphanaacademy/em-consumer

Along with their full Youtube-playlist about SAP Cloud Platform Enterprise Messaging:
